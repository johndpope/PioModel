## 如何使用
### [Pio](https://github.com/Dreamer-Paul/Pio)
在 `Pio 设置` 中的 `自定义配置文件地址` 中输入本项目的资源链接。

### [Live2d](https://github.com/journey-ad/live2d_src)
{Canvas} 中填入 `画布(canvas)ID`。  
{URL} 中填入本项目的资源链接。  
```javascript
loadlive2d("{Canvas}", "{URL}");
```
